## Описание Проекта

Компилятор **AAR-2025** представляет собой транслятор для специализированного учебного языка программирования. Основная цель проекта — разработка полной цепочки компиляции: от лексического анализа до генерации исполняемого кода.

Компилятор преобразует исходный код на языке AAR-2025 в код ассемблера для платформы **x86** (синтаксис MASM) с последующей компиляцией в исполняемый файл Windows.

## Спецификация Языка AAR-2025

Язык AAR-2025 был разработан с учетом следующих ограничений и возможностей:

### Типы Данных

  * **Целочисленный (`int`):** 2 байта.
  * **Строковый (`string`):** Динамическая строка.
  * **Логический (`bool`):** Для условных операций.

### Арифметические Операции

Поддерживается полный набор стандартных арифметических операций:

  * Сложение (`+`)
  * Вычитание (`-`)
  * Умножение (`*`)
  * Деление (`/`)
  * Остаток от деления (`%`)

### Управление Потоком

  * **Условный оператор (`if-else`):** Для выполнения ветвлений на основе логического условия.
  * **Оператор выбора (`switch-case`):** Позволяет выбрать один блок кода из множества возможных значений.

### Функции Стандартной Библиотеки

  * **Возведение в степень**
  * **Произвольная функция** 

### Представление Литералов

  * Целые числа поддерживают **шестнадцатеричное** и **двоичное** представление.
  * Логические литералы (`true`, `false`) представлены произвольно.

## Структура Компилятора

Транслятор реализован на языке **C++20** в среде **Visual Studio 2020** (ОС Windows 11, 64-bit).

Компилятор состоит из следующих функциональных модулей:

1.  **Лексический Анализатор (Lexical Analyzer):** Разделение исходного кода на токены и формирование Таблицы Лексем (LT) и Таблицы Идентификаторов (IT).
2.  **Синтаксический Анализатор (Syntax Analyzer):** Проверка грамматики языка и построение **Графа дерева разбора** (для графической части проекта).
3.  **Семантический Анализатор (Semantic Analyzer):** Проверка типов и корректности использования идентификаторов.
4.  **Генератор Кода (Code Generator):** Преобразование внутреннего представления в код **Ассемблера x86 (MASM)**.

## Сборка и Запуск

### Требования

  * Операционная система **Windows 10/11 (64-bit)**.
  * Среда разработки **Visual Studio 2020** или более новая.
  * Установленные инструменты для работы с ассемблером (например, **MASM32** или интеграция с `ml.exe` и `link.exe`).

### Использование

Компилятор запускается из командной строки со следующими параметрами:

```bash
> Compiler.exe -in:<имя_файла_входа> -out:<имя_файла_выхода.asm> -log:<имя_файла_лога>
```

| Параметр | Описание |
| :--- | :--- |
| `-in:<путь>` | Указывает путь к **входному файлу** с исходным кодом AAR-2025. |
| `-out:<путь>` | Указывает путь для генерации **ассемблерного файла** (`.asm`). |
| `-log:<путь>` | Указывает путь для сохранения **протокола работы** транслятора. |

### Пример Запуска

```bash
> Compiler.exe -in:..\test\example.txt -out:..\bin\output.asm -log:..\log\compiler.log
```
